## Wat werkt cross-pagina — probeer dit zelf

1. Open `index.html`
2. Klik **Frankrijk-card** in de Landen-sectie → `country.html`
3. Klik **Lyon prominent** (21:9) → `city.html`
4. Klik **Cour des Loges** in de Sanctuary-sectie → `entry.html`
5. Op de entry:
   - Klik de grote bookmark "Bewaar in atlas"
   - Klik de 3 status-pills (Wishlist / Visited / Favorite)
   - Typ een notitie in het tekstveld
6. Open `atlas.html`
7. Zie alles dynamisch verschijnen:
   - Cour des Loges als pulse-pin op de Frankrijk-kaart
   - In de "Bewaard"-grid met status-badges
   - In de 3 status-secties (Wishlist/Visited/Favorite)
   - In het notitie-overzicht
8. Klik bookmark op een atlas-card → verwijdert uit atlas
9. Terug naar entry → bookmark is leeg

Theme-toggle (rechtsboven) is persistent over alle pagina's via `fm-theme` localStorage key.

---

## LocalStorage keys

| Key | Type | Inhoud |
|---|---|---|
| `fm-theme` | string | `'light'` of `'dark'` |
| `fm-saved-entries` | JSON array | slugs van bewaarde entries |
| `fm-status-{slug}` | JSON object | `{wishlist, visited, favorite}` per saved entry |
| `fm-note-{slug}` | string | privé notitie per saved entry (auto-save 400ms debounce) |

Op `atlas.html` zit een hardcoded `ENTRIES` registry met alle 40 slugs uit city/entry/country — slug → `{name, cat, city, country, district, lat, lng, image, href}`.

## Beslissingen die in steen staan (uit CLAUDE.md)

- **Light-first** — `#f4f2ef` als bg, dark via toggle
- **Eén accent** — taupe `#afa297`, géén accent-rood, géén success-groen
- **Fonts** — Fraunces (serif) + Inter Tight (sans). Géén Bebas, géén monospace UI
- **Border-radius 0** op alle UI-componenten
- **Easing** — `cubic-bezier(0.16, 1, 0.3, 1)`
- **Iconen** — Iconify Solar Linear, `stroke-width="1.5"`
- **Géén PB-familie tricks** — glass-morfisme, shimmer, scanlines, heartbeat glow, flashlight, corner marks (die horen bij DMM/PerfectMoods)
- **Eén Entry-type** met verplichte lat/lng. Place-hiërarchie (Country → Region → City → District) afgeleid uit coords, niet handmatig
- **City-promotion** — ≥ 5 entries én ≥ 2 categorieën
- **Member day-1** — Save, Lists, Status, Notes
- **AI** schrijft Country/Region/City-intro's en "in de buurt"-blokken — reviewable, niet auto-publiceren
